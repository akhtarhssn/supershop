# E-Commerce Frontend

A high-performance, modern storefront for the E-Commerce Multi-Vendor Platform, built with Next.js 16.

## 🎨 Design Philosophy

- **Premium Aesthetics**: Clean, responsive layout with smooth transitions.
- **Dynamic Interaction**: Hover effects, micro-animations, and interactive components.
- **State Management**: Robust handling of carts, user sessions, and store data using Redux and Zustand.
- **Form Handling**: Seamless validation and error reporting.

## 🚀 Getting Started

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```
2. Set up environment variables:
   Create a `.env.local` file:
   ```env
   NEXT_PUBLIC_API_URL=http://localhost:5000/api/v1
   ```

3. Run in development:
   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack

- **Framework**: Next.js 16
- **UI Components**: Shadcn/UI & Radix UI
- **Styling**: Tailwind CSS 4
- **State**: Redux Toolkit & Zustand
- **Animations**: GSAP
- **Table Management**: TanStack Table

## 📁 Key Directories

- `src/app`: Page routes and layouts.
- `src/components`: Reusable UI elements.
- `src/redux`: Global state logic.
- `src/types`: Shared TypeScript interfaces.
